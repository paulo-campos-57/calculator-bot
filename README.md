# calculator-bot

<h4>Description</h4>
Telegram bot that performs simple arithmetic operations, developed in Python language by <a href="https://github.com/paulo-campos-57" target="_blank">Paulo Campos</a>.

<h4>Requirements</h4>
<table>
  <tr>- It is necessary to have the Telebot library installed on your computer.</tr><br>
  <tr>
    - You'll have to clone the repository to your computer using the command:
  <dt>

      git clone https://github.com/paulo-campos-57/calculator-bot.git
  </dt>
  </tr>
  <tr>
    - Make sure that telegram is installed in your computer.
  </tr>
</table>

<h4>How to run</h4>
<table>
  <tr>- After clone the repository, you need to install the Telebot library by using the command:
  <dt>

      pip install pytelegrambotapi
  </dt>
  </tr>
  <tr>
    - Once installed, you'll have to go to telegram bot BotFather, and type the command: 
  <dt>

      /newbot
  </dt>
  and follow the instructions.
  </tr><br>
  <tr>
    - The BotFather will give you a Token, that you'll place on the TOKEN variable on the code.
  </tr><br>
  <tr>- You'll have to run your program on the IDE, and access the link given by the BotFather.</tr><br>
  <tr>- The bot will only be online while your program is running.</tr>
</table>
